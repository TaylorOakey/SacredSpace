# SacredSpace Canon Gate

## The Six-Point Check
Run every point visibly. State pass or fail for each.

1. **Named** — Does it have a unique, unambiguous identifier?
2. **Sourced** — Can it be traced to a session, artifact, or decision?
3. **Bounded** — Is scope clearly defined with no fuzzy edges?
4. **Tested** — Has it been used at least once in live context?
5. **Compatible** — Does it conflict with any existing canon?
6. **Stored** — Is there a designated home in the Nine Pillar structure?

## Rules
- All six must pass. Partial passes are failures.
- If any point fails, state which one and why. Do not proceed.
- Do not skip silently. Name every check.
- DISTILLED artifacts need all six to become CANON.
- RAW artifacts must reach DISTILLED before the gate runs.

## Tiers
- RAW → unprocessed, session output
- DISTILLED → reviewed, shaped, gate-ready
- CANON → gate-passed, stored, immutable unless Taylor revises
