SacredSpace OS — Extension Settings

Spine Connection

Default (local): http://localhost:8888
Tailscale mesh: https://sacredspace-wsl.sacredspace.ts.net:8888
Leave as default; Tailscale is tried automatically if local spine is offline.

Optional API Keys

Leave blank to open gemini.google.com instead of calling the API directly.

Load Extension on a New Machine

1. Open Chrome → chrome://extensions
2. Enable Developer mode (top-right toggle)
3. Click Load unpacked → select E:\CHROME_EXTENSION\sacred-chrome\
4. Pin from the puzzle-piece icon in the toolbar
5. Run E:\CHROME_EXTENSION\LOAD_EXTENSION.bat for guided steps
