## 3. Then: call it from an app
Here’s a **super minimal Flutter-style flow** (not full app, just the shape):

```dart
import 'package:flutter/material.dart';
import 'package:http/http.dart' as http;
import 'dart:convert';

void main() => runApp(SacredApp());

class SacredApp extends StatelessWidget {
  @override
  Widget build(BuildContext context) {
    return MaterialApp(
      home: CurriculumHome(),
    );
  }
}

class CurriculumHome extends StatefulWidget {
  @override
  _CurriculumHomeState createState() => _CurriculumHomeState();
}

class _CurriculumHomeState extends State<CurriculumHome> {
  String result = "Choose a root to get your task.";

  Future<void> getTask(String root) async {
    // replace with your Google AI Studio endpoint + API key
    final url = Uri.parse("https://generativelanguage.googleapis.com/v1beta/models/gemini-1.5-flash:generateContent?key=YOUR_API_KEY");

    final body = {
      "contents": [
        {
          "parts": [
            {"text": "I am working in the $root root. I have 30 minutes. Give me my next task."}
          ]
        }
      ]
    };

    final res = await http.post(
      url,
      headers: {"Content-Type": "application/json"},
      body: json.encode(body),
    );

    final data = json.decode(res.body);
    setState(() {
      result = data["candidates"][0]["content"]["parts"][0]["text"];
    });
  }

  @override
  Widget build(BuildContext context) {
    return Scaffold(
      appBar: AppBar(title: Text("Sacred Creator")),
      body: Padding(
        padding: const EdgeInsets.all(16.0),
        child: Column(
          children: [
            Wrap(
              spacing: 8,
              children: [
                ElevatedButton(onPressed: () => getTask("Mind"), child: Text("🧠 Mind")),
                ElevatedButton(onPressed: () => getTask("Heart"), child: Text("❤️ Heart")),
                ElevatedButton(onPressed: () => getTask("Hands"), child: Text("✋ Hands")),
                ElevatedButton(onPressed: () => getTask("Spirit"), child: Text("🌀 Spirit")),
              ],
            ),
            const SizedBox(height: 24),
            Expanded(child: SingleChildScrollView(child: Text(result))),
          ],
        ),
      ),
    );
  }
}
```

What this does:
- shows 4 buttons (your 4 roots),
- when you tap one, it calls Gemini (Google AI Studio) with your context,
- it shows you the task.

That’s already an **interactive curriculum app.**

## 4. Put it on your home screen
