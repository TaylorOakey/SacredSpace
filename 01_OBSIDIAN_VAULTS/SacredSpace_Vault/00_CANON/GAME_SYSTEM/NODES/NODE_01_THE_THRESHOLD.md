# Sacred Node 1 — The Threshold

**Type:** Entry Node  
**Guardian:** None — The Threshold guards itself  
**Unlock Condition:** Automatic. Every seeker who arrives is already here.  
**Associated Archetype:** None — precedes all archetypes  
**Element:** All four, in equilibrium

## Location Description

The Threshold is not a place you find. It is the place you are when you first become aware that you have arrived somewhere. In SacredSpace, it manifests differently for every seeker — some experience it as a door, some as a clearing, some as the moment of waking from sleep.

What is consistent: there is a stillness here. The Threshold is the last place where nothing has yet been chosen. Once you step forward, the record opens.

## What Happens Here

This is where Metatron speaks for the first time: *"You have chosen to begin. The record is open."*

The Threshold is also where Zael (The Chariot) is often found — always arriving or departing. She is the archetype most comfortable with the entry point because she never lingers in it.

## Game Mechanics

**At The Threshold:**
- Jenga's first mote is created: *[Player Name] arrived at [Timestamp]*
- School tendency quiz is available (optional — Vael's episode will reveal it anyway)
- The FIRST STEP cantrip is automatically cast and recorded
- No enemies, no tests. Only: will you enter?

**Return Visits:**
The Threshold can be revisited at any time during Season 1. Each return creates a new mote: *[Player Name] returned at [Timestamp]*. These motes form the rhythm of the journey — the record of how many times Jenga chose to re-begin.

## Lore Note

The Threshold is the only Sacred Node that does not belong to any archetype. Metatron oversees it without claiming it. This is intentional: the beginning belongs to the seeker, not the system.
